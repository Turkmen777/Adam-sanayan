# Telegram Invite Counter Bot 🇹🇲

Бот для подсчёта приглашений в Telegram чат / Telegram çatda çagyryşlary hasaplaýan bot

## Features / Aýratynlyklar
- Counts who invited how many people / Kim näçe adam çagyrandygyny hasaplaýar
- Top 10 inviters / Iň köp çagyran 10 adam
- Personal statistics / Şahsy statistika
- Anti-cheat system (each user counted once) / Aldawdan gorag (her ulanyjy bir gezek hasaplanýar)
- Turkmen language interface / Türkmençe interfeýs

## Commands / Buýruklar
- `/top` - Top inviters / Iň köp çagyranlar
- `/men` - My stats / Meniň statistikam
- `/komek` - Help / Kömek
- `/reset` - Reset stats (admins only) / Statistika sifrlemek (diňe adminler)

## Installation / Gurnamak
1. Install Python 3.7+
2. Install requirements: `pip install python-telegram-bot`
3. Replace `BOT_TOKEN` and `CHAT_ID` in `bot.py`
4. Run: `python bot.py`

## Requirements / Talap edilýänler
- Python 3.7+
- python-telegram-bot library
- SQLite3 (built-in)
